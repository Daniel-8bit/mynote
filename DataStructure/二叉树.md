## DFS 遍历（前，中，后）
三者的递归算法比较简单，下面的是迭代算法，都是用栈实现

1. 前序遍历

```java
    public List<Integer> preorderTraversal(TreeNode root) {

        List<Integer> res = new ArrayList<>();
        Deque<TreeNode> s = new LinkedList<>();

        if(root == null) return res;

        while(root != null || !s.isEmpty()) {
            while(root != null) {
                res.add(root.val);
                s.push(root);
                root = root.left;
            }
            root = s.pop();
            root = root.right;
        }
        return res;
    }
```
2. 中序遍历

```java
    public List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> res = new ArrayList<>();

        //将左孩子放入栈中
        Deque<TreeNode> s = new LinkedList<>();
        while(root != null || s.peek() != null) {
            while(root != null) {
                s.push(root);
                root = root.left;
            }
            root = s.pop();
            res.add(root.val);
            root = root.right;
        }
        return res;
    }
```
3. 后序遍历(前右左+反转)

```java
  public List<Integer> postorderTraversal(TreeNode root) {
        List<Integer> res = new ArrayList<>();
        if(root == null) return res;

        Deque<TreeNode> s = new LinkedList<>();
        while(root != null || !s.isEmpty()) {
            while(root != null) {
                res.add(root.val);
                s.push(root);
                root = root.right;
            }
            root = s.pop();
            root = root.left;

        }
        Collections.reverse(res);
        return res;
    }
```
